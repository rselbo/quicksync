install.nsi is a Nullsoft SuperPIMP install script.
NSIS is a free installer scripting system, read more here http://nsis.sourceforge.net/Main_Page

Remember to copy the following files into this folde
QtCore4.dll
QtGui4.dll
QtNetwork4.dll

If the default ignore rules have changed update ignore reg

If any new files need to be installed edit install.nsi
